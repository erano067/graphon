# Graphon

**High-performance graph visualization library for React**

Graphon is a powerful, WebGL-powered graph visualization library built on PixiJS. It provides smooth, interactive graph rendering with support for large datasets, custom styling, and advanced features like clustering and animations.

## Quick Start

```tsx
import { Graphon } from '@graphon/react';

function MyGraph() {
  const nodes = [
    { id: 'a', data: { label: 'Node A' } },
    { id: 'b', data: { label: 'Node B' } },
    { id: 'c', data: { label: 'Node C' } },
  ];

  const edges = [
    { id: 'ab', source: 'a', target: 'b', data: {} },
    { id: 'bc', source: 'b', target: 'c', data: {} },
  ];

  return (
    <Graphon
      nodes={nodes}
      edges={edges}
      width={800}
      height={600}
      onNodeClick={(event) => console.log('Clicked:', event.node.id)}
    />
  );
}
```
